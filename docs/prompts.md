# 提示词模板

## 功能任务模板

```text
请用最小且安全的改动集完成下面这个任务。

任务：
{{TASK}}

约束：
- 保持现有代码模式
- 避免无关改动
- 只在必要时更新测试
- 总结 touched files、tests run 和 remaining risks
```

## 调试任务模板

```text
请调查并修复下面这个 bug。

问题：
{{BUG}}

要求：
- 先识别根因
- 优先采用最小且安全的修复
- 如有必要，补充或更新回归覆盖
- 总结原因、修复方式和验证结果
```

## 建议用法

- 把 `{{TASK}}` 或 `{{BUG}}` 替换成具体上下文
- 对高风险任务补充“哪些内容不要改”
- 让工具在输出里明确列出 touched files、tests run、remaining risks
